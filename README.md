## Getting Started

### What is Unicorn

It's a search for the perfect seed. Checkout http://github.com/lukeheath/unicorn.

### What is the Unicorn Generator?

It creates unicorns.

Install the generator:

```bash
yo install generator-unicorn
```

Run the generator:

```bash
yo unicorn
```
